Политика обработки персональных данных

Настоящая Политика определяет порядок обработки и защиты персональных данных, предоставляемых пользователями на сайте shevgruz.ru.

1. Общие положения

1.1. Владелец сайта shevgruz.ru (далее — "Оператор") соблюдает Федеральный закон РФ от 27.07.2006 №152-ФЗ "О персональных данных".

2. Обрабатываемые данные

2.1. Мы собираем следующие данные: имя, номер телефона, адрес электронной почты, комментарии и другую информацию, введённую в формах сайта.

3. Цель обработки данных

  • Оказание услуг по перевозке;
  • Консультации и обратная связь;
  • Маркетинговые уведомления (при согласии пользователя).

4. Согласие пользователя

4.1. Заполняя любую форму на сайте, пользователь выражает согласие на обработку своих персональных данных.

4.2. Чекбокс согласия обязателен для отправки форм.

5. Хранение и передача

5.1. Данные не сохраняются на сервере сайта и передаются напрямую через API мессенджера Telegram в зашифрованном виде.

6. Меры защиты

  • Используется HTTPS (SSL);
  • Отсутствие хранения данных снижает риски утечки;
  • Сайт не использует cookies для отслеживания персональных данных.
